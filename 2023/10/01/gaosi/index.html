高斯分布与机器学习

概率
数学
发布日期:   2023-10-01
文章字数:   730
阅读时长:   2 分

正态分布，也被称为高斯分布，是一种常见的连续概率分布。它的图形形状为钟形曲线，是一种非常重要且广泛使用的概率分布。

正态分布的数学表达式

正态分布的概率密度公式为：

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{ -\frac{(x-\mu)^2}{2\sigma^2} }
$$

其中，$\mu$ 是均值，$\sigma$ 是标准差。这个函数描述了在给定均值和标准差的情况下，一个随机变量取某个值的概率。

  • 大约68%的数据位于均值的一个标准差范围内
  • 大约95%的数据位于两个标准差范围内
  • 大约99.7%的数据位于三个标准差范围内

如何用python生成标准正态分布数据

numpy中有一个random模块，它包含了生成随机数的各种方法。其中一个方法就是random.randn，它可以返回一个或多个服从标准正态分布（均值为0，方差为1）的随机数。这在很多领域都有应用，比如统计学、机器学习、信号处理等。

random.randn函数的语法如下：

numpy.random.randn(d0, d1, ..., dn)

其中，d0, d1, …, dn是可选的整数参数，表示返回数组的维度。如果没有提供任何参数，那么函数会返回一个单独的Python浮点数。如果提供了多个参数，那么函数会返回一个形状为(d0, d1, …, dn)的数组，数组中的元素都是从标准正态分布中随机采样的浮点数。

一维标准正态分布数组示例： 

>>> import numpy as np
>>> np.random.randn(5)
array([-0.51733692,  0.48813676, -0.88147002,  1.12901958,  0.68026197])

二维标准正态分布数组，包含3行4列的随机数：

>>> import numpy as np
>>> np.random.randn(3, 4)
array([[ 1.33262386, -0.88922967, -0.07056098,  0.27340112],
       [ 1.00664965, -0.68443807,  0.43801295, -0.35874714],
       [-0.19289416, -0.42746963, -1.80435223,  0.02751727]])

random.randn 函数应用

假设我们有一只股票，其初始价格为100元，每天的价格变化服从均值为0，标准差为2的正态分布。
想要模拟这只股票在100天内的价格走势, 代码如下：

# 导入numpy和matplotlib库
import numpy as np
import matplotlib.pyplot as plt

# 设置初始价格和天数
initial_price = 100
days = 100

# 生成服从标准正态分布的随机数
random_numbers = np.random.randn(days)

# 计算每天的股票价格
stock_prices = initial_price + 2 * random_numbers

# 绘制股票价格走势图
plt.plot(stock_prices)
plt.xlabel('Day')
plt.ylabel('Price')
plt.title('Stock Price Simulation')
plt.show()

注意

  • random.random 是生成[0.0, 1.0)区间的一个随机小数, 而且只有一个参数，当需要生成多维数组时应该输入(m,n,…)：
    random.random((3,4))
  • random.rand 是生成[0.0, 1.0)区间服从均匀分布的随机小数
    random.rand(3,4)
